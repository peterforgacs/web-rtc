WebRTC demo
===================


Simple WebRTC demo using Node Js and WebSockets. 

----------


<i class="icon-file"></i> Install
-------------

```sh
$ git clone https://github.com/peterforgacs/webRTC
$ npm install -g node-static
$ npm install ws
$ static&
$ node server
```
